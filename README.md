# KalaStatic Examples

This demonstrates a few [KalaStatic](http://github.com/kalamuna/kalastatic) examples.

## Examples

- [Hello World](helloworld)

## Usage

In each example:

1. Change to the directory
    ```
    cd helloworld
    ```

2. Install the node modules
    ```
    npm install
    ```

3. Start KalaStatic
    ```
    npm start
    ```
